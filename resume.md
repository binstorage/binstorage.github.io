## 자기소개


**"그냥 해봐 할 수 있어!"** 가장 제가 좋아하는 말입니다.

현재 제로트러스트를 기반으로 하는 네트워크 보안 회사에서 iOS, macOS 환경의 네트워크 보안 에이전트를 개발하고 있습니다. 처음에는 Swift, UIKit, RxSwift 기술 스택만 보유하고 있었으나, iOS/macOS 공통 UI 프레임워크를 사용하기 위해 SwiftUI를 학습하고, 이어서 Combine, Extension, Daemon, XPC, App Group, 네트워크 통신, VPN, 보안 등에 대해 공부하며 프로젝트를 수행했습니다.

비즈니스 로직의 핵심인 패킷 제어를 위해 C++ 기반 네트워크 모듈을 Swift에서 호출해야 했고, 이를 위해 C++, Objective-C, Objective-C++의 호환성에 대해 깊이 있게 학습했습니다. 단순히 사용법을 아는 것에 그치지 않고, 컴파일러와 링킹 단계에서의 처리 과정까지 이해하려 노력합니다.

모듈화와 아키텍처 설계에 관심이 많아 사이드 프로젝트에서 Tuist, Clean Architecture, TCA를 먼저 적용해보고, 이를 회사 프로젝트에도 제안하여 도입했습니다. 대규모 서비스에도 관심이 있어 카카오뱅크 등의 사례를 참고하며 RIBs, ReactorKit 등을 학습하고 있습니다.
